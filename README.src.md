# MlDict
Program for search in dictionary database.

## Owerview

### Base execution

```
#exec dict help
```

### Command list

```
#exec dict help list
```

### Command search

```
#exec dict help search
```

### Example usage

```
$ dist search -e hello
#exec dict search hello
```
